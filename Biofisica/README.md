# Biofísica

Programas en Python orientados al análisis de datos mediante técnicas de aprendizaje automático (machine learning) aplicadas a problemas relacionados con datos biológicos o biomédicos.

## 📂 Contenido

* Árbol de decisión: utilizados para tareas de clasificación y análisis de patrones en conjuntos de datos
* Regresión lineal: modelos utilizados para estudiar relaciones entre variables y realizar predicciones a partir de datos
