# 📚Proyectos de Programación - Grado y Máster
Este repositorio recopila distintos ejercicios y proyectos de programación realizados durante el Grado en Física y el Máster en Matemática Industrial. Incluye proyectos en Python, MATLAB, Fortran, C++ y Java relacionados con distintos ámbitos de la computación científica, la programación y los métodos numéricos. 

##  📂Contenido del repositorio

| Asignatura | Lenguajes | Contenido |
|-----------|-----------|-----------|
| Biofísica   | Python    | Aplicaciones de machine learning: árbol de decisión y regresión lineal     |
| Ecuaciones diferenciales ordinarias   |  MATLAB   | Métodos lineales multipaso, método theta y Euler explícito    |
| Física computacional   | Fortran    |  Asignatura básica de Python: raices de ecuaciones, sistemas lineales, tratamiento de datos, derivación, integración, números aleatorios y ecuaciones diferenciales  |
| Métodos numéricos estocásticos   |  MATLAB   | Generación de muestras de distintas distribuciones y resolución de ecuaciones diferenciales estocásticas mediante Euler-Maruyama y Milstein I |
| Métodos numéricos y programación   | MATLAB   | Factorización QR |
| Programación en C++  |  C++   | Programación orientada a objetos en C++ |
| Redes de computadores y computación distribuida   | Java    | Creación de un servidor y cliente de eco y aproximación de pi mediante el método Monte Carlo   |
| Simulación en física de materiales   |  Fortran   | Programa que crea la red de un material, lo hace evolucionar hasta el equilibrio y calcula algunas propiedades físicas de dicho material |

Cada carpeta se corresponde con una asignatura del máster o del grado.

## 🎯Objetivo

Este repositorio sirve como portfolio técnico para roles relacionados con:

* Modelación matemática y simulación.
* Data science aplicado.
* Matemática aplicada a la industria.
